# TemplateWaylandCompositor

A template project for creating a Wayland Compositor using Qt6 and QML.

## Prerequisites

- Qt 6.8 or higher
- CMake 3.16 or higher
- C++23 compatible compiler (GCC, Clang)
- Wayland development libraries

## Build Instructions

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Running

```bash
./appTemplateWaylandCompositor
```

## Project Structure

- `src/`: Source code (C++ and QML)
- `CMakeLists.txt`: Build configuration
